# 指静脉认证平台 接口源码

## 目录说明

+ EVA   
    |-api-web : 单体工程整合层,可将任意api-*工程接入统一发布   
    |-addon-generator : 代码生成器   
-----------------------------------------------------------------   

 - 连接池: Hikari
 - 配置加密: Jassypt
 - SQL监控: P6SPY
 - 数据库版本管理: Flyway
 - JSON: Fastjson
 - 日志: Logback
 - 工具包: Hutool
 - ORM: Mybatis, Mybatis-plus
